Server setup

Hoi! Lijkt erop dat dit een nieuwe server is. Welkom bij Ferox Hosting!

In deze setup wizard gaan we je helpen jouw server zo snel mogelijk aan de gang te krijgen, zonder enige benodigde kennis.

Ben je er klaar voor? Klik dan op Volgende.



Let op: Gemaakte keuzes en wijzigingen worden pas op beschikbaarheid gecontroleerd en toegepast na het volledig afronden van de setup. Bij tussentijds stoppen gaan keuzes dus verloren!

Minecraft EULA

Om je Minecraft server op te kunnen starten moet je akkoord gaan met de Mojang's End User Licence Agreement (" EULA "), die te vinden is op https://account.mojang.com/documents/minecraft_eula. Hoofdzakelijk betekent dit dat je geen items mag verkopen die mensen een oneerlijke voorsprong geven.


Door op Volgende te klikken en de rest van deze Setup te gebruiken ga je akkoord met de Minecraft EULA.

Versie

Er zijn veel verschillende Minecraft server versies beschikbaar. Vanilla servers, servers met plugins, servers met mods.. Misschien heb je er al een in gedachten, misschien niet. Neem een kijkje op ons kennisbank artikel voor meer informatie en maak je keuze.


Voor de meeste gebruikers is Paper(Spigot) aangeraden.

Voor survival modes is de nieuwste versie aangeraden. Voor games is 1.12.2 vaak een betere keuze.

Als je wil dat spelers op nieuwere versies ook kunnen spelen kan je de plugin ViaVersion installeren naar het plugins mapje in de bestandsbeheerder.

OPs (Operators)

Zekere spelers zijn misschien admin of mod op jouw server, en moeten om hun taak uit te kunnen voeren administratieve commands gebruiken. Spelers met OP (operator) status kunnen dit. Voeg hier alle gebruikersnamen van spelers toe die OP moeten hebben op jouw server.

Whitelist

Soms wil je dat alleen specifieke spelers op jouw server kunnen spelen. Zo ja, zet dan de whitelist aan en voeg alle gebruikernamen van spelers toe die op jouw server moeten kunnen inloggen en spelen.


Sta alleen spelers op de whitelist toe om te joinen



Spelers op de whitelist:


Het is mogelijk om later ook nieuwe gebruikers te whitelisten met het commando whitelist add degebruikersnaam of de whitelist uit te zetten met whitelist off

Subdomeinen

Het IP van jouw server is 45.136.141.71:20356. Dit is wat spelers in Minecraft in kunnen vullen om met jouw server te verbinden.

Een subdomein is vaak makkelijker te onthouden. Je kunt 2 gratis <keuze>.ferox.host of <keuze>.feroxhosting.nl subdomeinen aanmaken en dan kunnen spelers ook via dit adres met jouw server verbinden.


  • .feroxhosting.nl
  • .ferox.host


Wil je een adres zonder .ferox.host of .feroxhosting.nl? Het is ook mogelijk om een volledig eigen naam te bestellen op onze site. Deze zal binnen enkele uren automatisch werken
